# Private Candidate Materials

This folder is intended for local candidate information.

Recommended local files:

- `candidate-evidence.md`
- `resume.pdf`
- `portfolio-notes.md`
- `career-stories.md`

These files are ignored by Git and must not be committed.

Do not place the following information in the public repository:

- Personal phone numbers
- Private email addresses
- Confidential company information
- Internal product names
- Unreleased product details
- Private portfolio passwords
- Referral-contact information
- Unapproved business metrics
